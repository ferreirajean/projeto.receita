# 🍳 Simple Omelette Recipe

Projeto de uma página web estática que apresenta uma receita simples de omelete, com layout moderno, responsivo e organizado utilizando HTML e CSS.

---

## 📸 Preview

Uma interface limpa e agradável com:

* Imagem da receita
* Descrição
* Tempo de preparo
* Ingredientes
* Modo de preparo
* Tabela nutricional

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* Google Fonts (Outfit e Young Serif)

---

## 📱 Responsividade

O projeto foi adaptado para dispositivos móveis utilizando **media queries**, garantindo:

* Ajuste automático de largura
* Melhor legibilidade em telas pequenas
* Layout fluido e centralizado
